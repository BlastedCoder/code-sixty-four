Bracket League App
A real-time, snake-draft tournament application built with React, Next.js, and Supabase.

Core Features
Dynamic Snake Draft: Automated 8-player rotation to roster all 64 tournament teams.

Real-time Leaderboard: Instant standings updates powered by Supabase Realtime and a 1-point-per-win scoring logic.

League Integrity: Automatic league locking after 64 picks to prevent mid-tournament roster changes.

Commissioner Tools: Full control over draft randomization, member invites, and league finalization.

Tech Stack
Frontend: React, Tailwind CSS, Lucide Icons.

Backend/Database: Supabase (PostgreSQL) with Row Level Security (RLS) and real-time listeners.
